Descrição Geral

Este sistema permite que o usuário adicione, liste e organize livros utilizando algoritmos eficientes de recomendação. A aplicação implementa estruturas de dados como árvores binárias para organizar os livros e facilitar operações avançadas de busca e recomendação. A capacidade máxima do sistema é de 100 livros.

1. Funcionalidades Principais

  • Adicionar Livros: Adiciona um novo livro ao acervo com informações como título, autor e ano de publicação.
  • Listar Livros: Exibe todos os livros cadastrados no sistema.
  • Ordenar Livros: Ordena os livros por título ou por autor.
  • Remover Livros: Remove um livro específico do acervo.
  • Recomendar Livros: Sugere livros relacionados com base em critérios específicos, como autor ou tema.

2. Interface Gráfica

A interface é desenvolvida com Swing em Java e contém:

  • Formulário de Adição: Campos para inserir informações sobre o livro.
  • Área de Exibição: Um painel que lista todos os livros cadastrados.
  • Botões de Ação: Opções para ordenar a lista de livros por título ou autor, remover livros e recomendar livros.

3. Algoritmos de Busca e Recomendação

Estrutura de Árvore (Árvore Binária de Busca):

A árvore binária de busca é utilizada para organizar os livros de forma eficiente, permitindo operações de busca rápidas e recomendações personalizadas com base nas relações entre autores e temas. A implementação atual inclui funcionalidades de adicionar, listar, remover e recomendar livros, além de métodos para buscar por autor e ordenar os livros.

4. Validações e Tratamento de Erros

O sistema valida a entrada do usuário para garantir que o ano de publicação seja um número inteiro. Caso contrário, uma mensagem de erro será exibida. Além disso, o sistema exibe um aviso quando o limite de livros é atingido.

5. Capacidade e Limitações

A aplicação suporta até 100 livros e utiliza estruturas de dados avançadas para organização e recomendação. Não há suporte para Bubble Sort na implementação atual, o que melhora a eficiência da ordenação.

6. Considerações Finais

Esta aplicação fornece uma interface amigável para gerenciamento de livros com funcionalidades práticas de recomendação e organização. Em versões futuras, funcionalidades como pesquisa avançada de livros e uma interface aprimorada para remoção de livros podem ser adicionadas para melhorar a experiência do usuário.

7. Classes e Métodos

Classe: ArvoreBinaria

Esta classe implementa uma árvore binária para gerenciar livros.

Métodos

  • inserir(Livro livro): Insere um novo livro na árvore.
  • exibirLivrosEmOrdem(): Exibe todos os livros na árvore em ordem.

Classe: GerenciadorDeLivros

Esta classe gerencia a adição, remoção e exibição de livros, bem como recomendações.

Métodos

  • adicionarLivro(String titulo, String autor, int ano): Adiciona um novo livro ao gerenciamento.
  • listarLivros(): Lista todos os livros no painel de exibição.
  • removerLivro(String titulo): Remove um livro com base no título.
  • buscarPorAutor(String autor): Busca livros de um autor específico.
  • ordenarPorTitulo(): Ordena os livros por título.
  • ordenarPorAutor(): Ordena os livros por autor.

Classe: Main

Classe principal que cria a interface gráfica do sistema de gerenciamento de livros.

Métodos

  • main(String[] args): Ponto de entrada do programa.
  • criarInterface(): Cria a interface gráfica do sistema.
