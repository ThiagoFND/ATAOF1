Sistema de Gerenciamento de Livros

Uma aplicação para organização e gerenciamento de livros, utilizando algoritmos de ordenação Quick Sort.

Descrição Geral

Este sistema permite que o usuário adicione, liste e organize livros utilizando algoritmos eficientes de ordenação. A aplicação implementa o Quick Sort para ordenar os livros por título e por autor, proporcionando uma forma prática e intuitiva de gerenciar o acervo. A capacidade máxima do sistema é de 100 livros.

1. Funcionalidades Principais

  • Adicionar Livros: Adiciona um novo livro ao acervo com informações como título, autor e ano de publicação.
  • Listar Livros: Exibe todos os livros cadastrados no sistema.
  • Ordenar Livros: Ordena os livros por título ou por autor usando o algoritmo Quick Sort.
  • Remover Livros: Remove um livro específico do acervo.

2. Interface Gráfica

A interface é desenvolvida com Swing em Java e contém:

  • Formulário de Adição: Campos para inserir informações sobre o livro.
  • Área de Exibição: Um painel que lista todos os livros cadastrados.
  • Botões de Ação: Opções para ordenar a lista de livros por título ou autor e remover livros.

3. Algoritmo de Ordenação

Ordenação (Quick Sort):

O Quick Sort é um algoritmo eficiente que utiliza a técnica de divisão e conquista. Ele seleciona um pivô e ordena a lista recursivamente, separando os livros em sublistas com base em suas comparações com o pivô.

4. Validações e Tratamento de Erros

O sistema valida a entrada do usuário para garantir que o ano de publicação seja um número inteiro. Caso contrário, uma mensagem de erro será exibida. Além disso, o sistema exibe um aviso quando o limite de livros é atingido.

5. Capacidade e Limitações

A aplicação suporta até 100 livros e utiliza o algoritmo Quick Sort para ordenação, que é eficiente para grandes volumes de dados. Não há suporte para Bubble Sort na implementação atual, o que melhora a eficiência da ordenação.

6. Considerações Finais

Esta aplicação fornece uma interface amigável para gerenciamento de livros com funcionalidades práticas de ordenação. Em versões futuras, funcionalidades como pesquisa de livros e uma melhor interface para remoção de livros podem ser adicionadas para aprimorar a experiência do usuário.
